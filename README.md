# reactTailwindPureaam

COMPLETE E COMMERCE WEBSITE




# HOW TO INSTALL

cd api 
npm  install

cd ui
npm install



# HOW TO START
cd api => npm start server will start at port 4000 (API)
cd ui => npm start react server will start at port 3000 (UI)
